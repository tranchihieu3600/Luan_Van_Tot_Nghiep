=====================================================================
HUONG DAN CAI DAT VA CHAY PROJECT LUAN VAN TREN MAY MOI
=====================================================================

Yeu cau truoc khi bat dau:
- May moi da duoc cai dat Python 3.8.10.
- Trong thu muc nay phai co file "requirements.txt" (danh sach thu vien).

BUOC 1: MO TERMINAL TAI THU MUC
- Mo thu muc "luanvan" tren may moi.
- Mo CMD, PowerShell hoac Terminal tai chinh thu muc nay. (Tren Windows co the go "cmd" vao thanh dia chi cua thu muc roi nhan Enter).

BUOC 2: TAO MOI TRUONG AO MOI
- Trong cua so hien ra, go lenh sau va nhan Enter:
  python -m venv venv

(Lenh nay se tao ra mot thu muc ten la "venv" de chua moi truong ao).

BUOC 3: KICH HOAT MOI TRUONG AO
- Neu ban dung Windows, go lenh:
  venv\Scripts\activate

(Thanh cong: Ban se thay chu "(venv)" hien ra o dau dong lenh).

BUOC 4: CAI DAT THU VIEN TU FILE REQUIREMENTS
- Dam bao da kich hoat moi truong ao (co chu venv o dau), go lenh:
  pip install -r requirements.txt

- Cho doi Terminal tai va cai dat cac thu vien. Khi bao "Successfully installed..." la hoan tat.

BUOC 5: CHAY CODE
- Bay gio moi truong da san sang, ban co the goi model local hoac chay file code chinh cua ban.
- Vi du: python full.py 

Luu y: Moi lan mo may len de lam viec hoac chay code, ban deu phai lam lai BUOC 3 de kich hoat moi truong ao roi moi chay code.
=====================================================================
